# Netskope App

## Tech Stacks
- react
- next
- mui
- axios
- mysql
- docker

## Launch App
- dockerhub
```
    docker pull kidd1118/netskope
    docker-compose up
```

- local docker
```
    docker-compose up
```

- local dev
```
    cd app
    npm run dev
```
