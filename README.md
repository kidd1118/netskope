# Netskope App

## Tech Stacks
- react
- next
- mui
- axios
- mysql
- docker

## Launch App
- docker
```
    docker-compose up
```
http://localhost:8081/


- local dev
```
    cd app
    npm run dev
```
http://localhost:3000/
